Telegram Bot-Currency Rate Checker

This bot provides real-time exchange rates. Just enter an amount, choose a currency pair or type your own (e.g., GBP/EUR), and get the result instantly.

How it works:

  • Enter a number (e.g., 100).
  • Choose a currency pair from the list or type one manually.
  • The bot returns the exchange rate and lets you enter a new value.

Tech stack: Python, Telegram Bot API, external exchange rate API.

To-Do List App

A simple task management application that allows you to add, view, update, and delete tasks. You can keep track of your tasks’ statuses (pending, in-progress, complete) and view descriptions for more details.

How it works:

  • Add tasks with a name and description.
  • View tasks in a list, with the option to update their status or delete them.
  • Easily manage and organize your tasks.

Tech Stack: Python, SQLite (for data storage).
